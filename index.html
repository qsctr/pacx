Intuitive UI

Our program is designed to have a highly visual user interface, with intuitive user interactions. Sending files with our program is as simple as dragging the file and dropping it on top of the widget.

Instead of using the typical, drab file folders used in traditional programs, our interface separates files into different modules which are easily distinguished. Over time our program will learn which modules are more useful and thus sort by relevance.

Modules is our biggest innovation. Traditional folders are not graphical, and do not provide any preview of the content within. Our modules are more aesthetically pleasing and provide more information of the folders within the module to allow users to find what they need faster.

Simple animations built into our application create a dynamic and aesthetically pleasing experience.

Convenient Organization

The organization of our application makes it perfect for usage in teams, both large and small. We provide the framework to organize your files instead of forcing you to create all of the folders. With Pacx, we place a large priority on saving time, no matter how little time we save. To do this, we allow you to instantly create the framework for the folders not by creating the folders than sharing them. Instead by simply sharing with your team members, we create the folder for you.

Inside each UI module are your personalized to-do lists, with assignment due-dates, notifications from widgets, and chats for communication. This obviously has great applications in schools like TAS.

Easy access

To access the program, simply click on the widget to get a quick view of what you need which includes the teams you are a part of, and the to-do lists from your various teams.

Notifications for file requests appear directly as an overlay on top of your work, while still remaining as minimally intrusive as possible.

Combine our desktop widget with pre-existing teams, by connecting your slack, google drive and trello. There is no need to create new account or sign-up with your email address.

Tasks from trello are put in your tasks module, chats from slack are put in either the chat or announcements module, and google drive appears in the file directory module.

Efficient file transfer

Many file transfer programs require long set-up, we minimize that process to make file sharing as painless as possible.

There is no need to search for a document someone once shared with you. Documents are sorted by relevancy and clicking on a user will bring up the documents that are shared between the two of you.

We allow users to request for files to be shared, instead of having to wait for their team members to share that file of their own volition.

Our desktop widget allows you to simply drag and drop to upload or share files.
